# 短路逻辑 && 与 ||
## && 短路逻辑
```js
A && B && C && D // 取第一个假值或 D，并不会取 true / false
```
```js
A && B // 同义于
if(A){
  B
}else{}
```
一般用法：
```js
A && B
fn && fn()
```
举例：
```js
console && console.log && console.log('hello')
```

## || 短路逻辑
```js
A || B || C || D // 取第一个真值或 D，并不会取 true / false
```
```js
A || B // 同义于
if(!A){
  B
}else{}
```
一般用法：
```js
A || B
A = A || B 
```
举例：
```js
a = a || 100 // 同义于
if(a){
  a = a
}else{
  a = 100 //保底值
}

```
